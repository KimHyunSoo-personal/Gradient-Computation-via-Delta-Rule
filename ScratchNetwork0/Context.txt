Delta rule을 이용하여 CNN의 weight gradient를 직접 구현해보았다.
Conv2d의 padding과 kernel size는 자유롭게 바꿔도 작동하지만 stride는 구현하지 못했다.
Feature Map도 확인해보고 싶어 해당 이미지를 실행파일 하위 폴더 Images에 저장하도록 만들었다. 매 batch의 첫번째 사진으로 돌렸을 때 feature map을 만들었다.

핵심
*matrix convolution의 gradient
X * W + b = Z (valid convolution)
dL/dW = dZ/dW * dL/dW = X * delta (valid convoultion)
dL/dX = dZ/dX * dL = W * flipped_delta (full convolition)
dL/db = dZ/db * dL/dZ = sum(delta)

*ReLU, MaxPool의 gradient
masking사용
ReLU: Forward에서 0이상의 값에 1 masking
MaxPool: Forward에서 max값과 동일한 값에 1 masking
